# 快速开始指南

## 🚀 立即开始

### 1. 安装依赖

```bash
npm install
```

这将安装以下关键依赖：
- `zustand` - 状态管理库
- `nanoid` - 唯一ID生成
- `@tauri-apps/api` - Tauri 核心 API
- `@tauri-apps/plugin-dialog` - 文件对话框
- `@tauri-apps/plugin-fs` - 文件系统操作

### 2. 运行开发环境

```bash
# 终端 1: 启动 Vite 开发服务器
npm run dev

# 终端 2: 启动 Tauri 应用
npm run tauri dev
```

### 3. 开始使用

应用启动后，您将看到：
- 左侧：项目树（显示 Frame 层级结构）
- 中间：画布（可视化编辑区域）
- 右侧：属性面板（编辑选中 Frame 的属性）
- 顶部：工具栏（快速操作按钮）

## 📖 基础操作

### 创建 UI 元素

1. 点击工具栏的 Frame 类型按钮（如 "Backdrop"、"Button"、"Text"）
2. 新的 Frame 将出现在画布中央
3. Frame 会自动添加到项目树中
4. 右侧属性面板显示 Frame 的详细信息

### 编辑 Frame

1. **选择 Frame**：
   - 在画布上点击 Frame
   - 或在项目树中点击 Frame 名称

2. **修改属性**：
   - 在右侧属性面板中编辑
   - 名称、类型、坐标、大小等
   - 修改会实时反映在画布上

3. **移动 Frame**：
   - 在属性面板中修改 X、Y 坐标
   - （拖拽功能待实现）

4. **调整大小**：
   - 在属性面板中修改宽度、高度
   - （调整手柄功能待实现）

### 画布操作

- **缩放画布**：
  - 按住 `Alt` 键 + 鼠标滚轮
  - 或点击右下角的 +/- 按钮

- **拖动画布**：
  - 按住 `Alt` 键 + 鼠标左键拖拽
  - 或按住鼠标中键拖拽

- **重置视图**：
  - 点击右下角的"重置"按钮

### 撤销和重做

- **撤销**：点击工具栏的 "↶ 撤销" 按钮
- **重做**：点击工具栏的 "↷ 重做" 按钮
- 快捷键（待实现）：Ctrl+Z / Ctrl+Y

### 组织结构

1. **更改父级**：
   - 选择要修改的 Frame
   - 在属性面板的"父级"下拉框中选择新的父级
   - Frame 会在项目树中重新组织

2. **相对定位**：
   - 勾选"相对于父级定位"选项
   - Frame 的坐标将相对于父级而不是画布

## 🎯 示例：创建一个简单的 HP 条

1. **创建背景**：
   - 点击工具栏的 "Backdrop" 按钮
   - 在属性面板中设置：
     - 名称：`HPBarBackground`
     - 宽度：`0.15`
     - 高度：`0.02`

2. **创建进度条**：
   - 点击工具栏的 "Backdrop" 按钮
   - 设置为背景的子元素
   - 在属性面板中设置：
     - 名称：`HPBar`
     - 父级：`HPBarBackground`
     - 相对定位：勾选
     - 文本颜色：设置为绿色

3. **添加文本**：
   - 点击工具栏的 "Text" 按钮
   - 设置为背景的子元素
   - 在属性面板中设置：
     - 名称：`HPText`
     - 父级：`HPBarBackground`
     - 文本：`100 / 100`
     - 水平对齐：居中
     - 垂直对齐：居中

## 🎨 UI 元素类型说明

### Backdrop（背景）
- 用途：静态背景、装饰
- 特点：可设置纹理

### Button（按钮）
- 用途：可点击的交互元素
- 特点：支持触发变量

### Text（文本）
- 用途：显示文本信息
- 特点：可调整颜色、缩放、对齐

### Checkbox（复选框）
- 用途：开关选项
- 特点：支持触发变量

### Horizontal Bar（进度条）
- 用途：显示数值进度
- 特点：支持最小值/最大值

## ⚙️ 通用设置

点击画布空白处或取消选择，右侧面板显示通用设置：

- **库名称**：生成代码时使用的库名
- **Origin 模式**：
  - Game UI：标准游戏 UI
  - World Frame：世界框架
  - Console UI：控制台 UI（1.32+）
- **隐藏默认 UI**：选择要隐藏的游戏默认 UI 元素

## 🐛 常见问题

### Q: 画布看不到 Frame？
A: 检查 Frame 的坐标是否在可视范围内（0-0.8, 0-0.6）

### Q: 无法选择 Frame？
A: 确保 Frame 的 Z 层级正确，上层 Frame 可能遮挡下层

### Q: 属性修改不生效？
A: 确保已按下回车键或点击其他地方使输入框失去焦点

### Q: 找不到某个功能？
A: 部分功能还在开发中，请查看 README_NEW.md 了解实现进度
